Running Ruby Code

3rdRail leverages standard Eclipse functionality to help you write Ruby code.

To run Ruby code in 3rdRail:

  1. Open the
    Ruby Perspective
    From the workbench select:
    Window Open Perspective Ruby Perspective
  2. Right-click the Ruby file in the Script Explorer.
  3. Select Run As Ruby Script.
    The file runs in the Console.
