Importing an Existing Rails Project

To import a Rails project:

  1. Open the
    Project Import Wizard
    From the workbench select:
    File Import Rails Import Existing Rails Project.
  2. Browse to the directory of the existing Rails project, select the project and click Open.
  3. Click Finish and the project opens.
