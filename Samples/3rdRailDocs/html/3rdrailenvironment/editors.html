Specialized Editors and Context-Aware Templates

3rdRail provides specialized editors that are based on context-aware templates. They provide support for scripting and coding in various languages.

For example, when you create a new RHTML file, the New RHTML File Wizard lets you choose a predefined template such as a basic form partial or a simple layout.

Access these editors from the File New menu:

  • RHTML Editor
  • RJS Editor
  • RXML Editor
