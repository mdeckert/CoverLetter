Configuring Preferences

Open Configuration Preferences

From the workbench select: Window Preferences

You can set these preferences for 3rdRail:

  • Rails Editors
  • Rails Consoles
  • Ruby Debug
  • Ruby Editor
  • Interpreters
  • Ruby Console
