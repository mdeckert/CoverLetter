Interactive Ruby

Interactive Ruby (IRB) functionality is available in the Ruby Console.

  • Code completion
  • Syntax highlighting

These websites have detailed information about the Ruby language:

  • Programming Ruby: the Pragmatic Programmer's Guide by Dave Thomas, with Chad Fowler and Andy Hunt
  • Ruby Programming From Wikibooks, the open-content textbooks collection
  • RubyLearning.com A Ruby Tutorial by Satish Talim

Ruby Documentation in 3rdRail

The standard Ruby language library contains more than 9,000 methods. To access Ruby Documentation in 3rdRail:

  • Mouse over object or method in the editor for context sensitive information.
  • Highlight an object or method in the editor and Press F2 for context sensitive information.
  • Open the Ruby Documentation from the Window Show View Ruby Documentation menu.
