Take a Tour of 3rdRail

Launch the Tour of the 3rdRail IDE to run it locally in the IDE on Mac, XP, or Vista. Depending on screen resolution, you may need to resize the IDE window to get better results.

Alternatively, you can get the web version.

The video is in Camtasia Studio Shockwave Flash format. It requires JavaScript to be enabled and the latest version of the Flash Player. If you are you using a browser with JavaScript disabled please enable it. If needed, update your version of the free Flash Player. If the video does not play correctly in your browser, see the Adobe Shockwave test page.
