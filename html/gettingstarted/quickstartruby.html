Quick Start: a New Ruby Project

For a brief overview of Ruby in 3rdRail, see this video: Introduction to the Ruby Console in 3rdRail.

If you are not familiar with the Eclipse framework, see: the Workbench tutorials.

To create a new Ruby project:

  1. If you have not already done so, configure the Ruby interpreter.
  2. If you are not already using it, switch to the Ruby Perspective.
    Window Open Perspective Ruby.
  3. Open the Ruby Project Wizard
    File New Project Ruby Ruby Project.
  4. Name the project and click Finish.
    The new Ruby project appears as an open project in the Script Explorer.
