Third Party Components

3rdRail bundles all the components you need to develop your application. The 3rdRail default components are:

  • Ruby VM 1.8.6-p111
  • Rails 1.2.6 and 2.0.2 runtimes
  • WEBrick application server, the default for Rails
  • MySQL database

3rdRail also provides:

  • InterBase Developer Edition
  • Mongrel application server

In addition, 3rdRail includes these plugins:

  • The Data Tools Platform
  • The Web Tools Platform
  • QuickREx
  • XPath Developer
  • Mylyn
  • Graysky Logwatcher
  • Subclipse Subversion Plugin
  • AJAX Toolkit Framework

For complete lists of bundled products and plugins, see Help About 3rdRail.

3rdRail System Requirements

3rdRail requires:

  • 32-bit Windows XP or Vista, or Mac OS X 10.4 or 10.5, or 32-bit Red Hat Enterprise Linux 4
  • 512 MB RAM minimum; 1GB recommended
  • 700 MB hard disk space minimum
  • 800 MHz minimum; 1.5GHz or faster recommended
  • The ability to download software directly from the internet
  • high-resolution monitor (1024x768 and 256 colors)

For updates to this information see the 3rdRail Technical Website.
